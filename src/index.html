About This Site


Full disclousure: I am not a front end developer. I dabble, but it's never really been my role. I built this website as a project to learn more about Terraform and serverless architecture.

As you can see from the site architecture, there's a surprising amount going on under the hood. Is it completely overengineered? Absolutely, but that's the whole point. The entire site is hosted for $0/month in AWS, is completely serverless and includes a REST API for the visitor counter in the footer below. Best of all, by using Terraform to represent my infrastructureas code, the site content and all underlying AWS infrastructure is versioned such that they can be rebuilt, tested, and deployed in seconds, via a push to my GitHub repo .
